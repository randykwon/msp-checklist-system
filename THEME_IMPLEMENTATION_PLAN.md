## 🎨 테마 상세 정보

### 🌙 야간 모드 (Night)
- **색상**: 다크 블루/그레이 계열
- **특징**: 기존 다크 테마 유지
- **용도**: 야간 작업, 눈의 피로 감소

### 🌊 바다 테마 (Ocean)
- **색상**: 블루 계열 그라데이션
- **특징**: 시원하고 차분한 느낌
- **용도**: 집중력 향상, 차분한 작업 환경

### ⛰️ 산 테마 (Mountain)
- **색상**: 그린 계열 그라데이션
- **특징**: 자연스럽고 안정적인 느낌
- **용도**: 장시간 작업, 안정감 제공

### 🌸 봄 테마 (Spring)
- **색상**: 핑크-퍼플 계열
- **특징**: 생기 넘치고 활기찬 느낌
- **용도**: 창의적 작업, 활력 증진

### ☀️ 여름 테마 (Summer)
- **색상**: 오렌지-레드 계열
- **특징**: 따뜻하고 역동적인 느낌
- **용도**: 에너지 충전, 활발한 작업

### 🍂 가을 테마 (Autumn)
- **색상**: 브라운-옐로우 계열
- **특징**: 따뜻하고 아늑한 느낌
- **용도**: 편안한 작업, 집중력 향상

### ❄️ 겨울 테마 (Winter)
- **색상**: 화이트-블루 계열
- **특징**: 깔끔하고 차가운 느낋
- **용도**: 명확한 사고, 정리된 작업 환경

## 🔧 기술적 특징

### CSS 변수 시스템
```css
:root {
  --theme-bg-primary: #색상값;
  --theme-text-primary: #색상값;
  --theme-ui-button-primary: #색상값;
  /* ... 기타 변수들 */
}
```

### 테마 매니저 클래스
```typescript
class ThemeManager {
  setTheme(themeId: ThemeType): void
  getCurrentTheme(): Theme
  subscribe(listener: Function): Function
  updateCSSVariables(): void
}
```

### React 컨텍스트 통합
```typescript
interface ThemeContextType {
  currentTheme: Theme;
  setTheme: (themeId: ThemeType) => void;
  availableThemes: Theme[];
}
```
